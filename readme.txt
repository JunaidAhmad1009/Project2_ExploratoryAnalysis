Project 2: Exploratory Data Analysis (EDA)

Objective
The goal of this project was to explore the cleaned dataset and generate insights using statistics, PivotTables, and charts.

Steps Performed
- Created summary statistics (COUNT, AVERAGE, MAX, MIN, STDEV)
- Built PivotTables for:
  - Order Status Analysis
  - Referral Source Impact
  - Monthly Sales Trend
- Designed charts:
  - Line Chart → Sales Trend Over Time
  - Bar Chart → Sales by Payment Method
  - Column Chart → Top Selling Products by Quantity
  - Scatter Chart → Relationship between Unit Price and Quantity
- Added final insights in a separate sheet

Files Included
- `Project2_ExploratoryAnalysis.xlsx` → Dataset + Statistics + Charts + Insights
- `/screenshots` → Images of PivotTables and charts

Tools Used
- Microsoft Excel (PivotTables, Charts, Statistical functions)

Outcome
A professional analysis report with dataset, statistics, visuals, and insights — ready for submission and portfolio showcase.
